# DEV-OPS-LR4 (Ansible Web Server Deployment) 
Этот проект содержит Ansible-плейбук, который автоматически устанавливает nginx на целевую виртуальную машину и копирует туда файл index.html для отображения веб-страницы.

## Содержимое репозитория
 - playbook.yml — основной Ansible-плейбук, выполняющий установку nginx и деплой index.html.  
 - inventory.conf — файл инвентаризации с адресом целевой виртуальной машины.  
 - index.html — шаблон веб-страницы, который копируется на целевой сервер.  

## Используемые технологии
 - Ansible — автоматизация конфигурации и деплоя.  
 - nginx — веб-сервер для отображения страницы.  
 - SSH — для удалённого подключения к целевой машине.  
 - VirtualBox + NAT — для настройки сетевого взаимодействия между виртуальными машинами.  

## Краткое описание шагов выполнения
1. Создана вторая виртуальная машина (копия первой) с новой сетевой конфигурацией (NAT, сеть 10.0.2.0/24).  
2. Настроен пользователь runner с правами sudo без пароля.  
3. Настроен SSH-доступ по ключу между машинами.  
4. Установлен Ansible на первой ВМ.  
5. Написан playbook.yml для установки nginx и копирования index.html.  
6. Плейбук успешно выполнен — на целевой машине развернут веб-сервер.
